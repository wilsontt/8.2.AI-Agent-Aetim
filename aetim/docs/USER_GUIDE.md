# 使用者手冊

## 概述

本文件說明 AETIM 系統的使用方式，包含功能說明、操作流程等。

## 快速開始

### 1. 登入系統

1. 開啟瀏覽器，訪問系統網址
2. 點擊「登入」按鈕
3. 系統會導向到身份驗證提供者（IdP）
4. 輸入帳號密碼登入
5. 登入成功後會自動導向到儀表板

### 2. 儀表板

儀表板顯示系統的整體狀態，包含：

- **系統健康狀態**：資料庫、Redis、AI 服務連線狀態
- **威脅統計**：威脅數量趨勢、風險分數分布、受影響資產統計、威脅來源統計
- **資產統計**：資產總數、受威脅影響的資產、對外暴露資產、高敏感度資產

## 功能說明

### 1. 資產管理

#### 1.1 檢視資產清單

1. 點擊左側選單「資產管理」
2. 查看資產清單
3. 使用搜尋和篩選功能查找特定資產

#### 1.2 新增資產

1. 點擊「新增資產」按鈕
2. 填寫資產資訊：
   - 主機名稱（必填）
   - 作業系統（必填）
   - 運行的應用程式（必填）
   - 負責人（必填）
   - 資料敏感度（必填）
   - 業務關鍵性（必填）
   - 是否對外暴露
3. 點擊「儲存」

#### 1.3 匯入資產

1. 點擊「匯入資產」按鈕
2. 選擇 CSV 檔案（≤ 10MB）
3. 預覽匯入資料
4. 確認後點擊「匯入」

#### 1.4 編輯資產

1. 在資產清單中點擊要編輯的資產
2. 點擊「編輯」按鈕
3. 修改資產資訊
4. 點擊「儲存」

#### 1.5 刪除資產

1. 在資產清單中選擇要刪除的資產
2. 點擊「刪除」按鈕
3. 確認刪除

### 2. 威脅管理

#### 2.1 檢視威脅清單

1. 點擊左側選單「威脅管理」
2. 查看威脅清單
3. 使用篩選功能查找特定威脅

#### 2.2 檢視威脅詳情

1. 在威脅清單中點擊要查看的威脅
2. 查看威脅詳細資訊：
   - 威脅標題與描述
   - CVE 編號
   - CVSS 分數
   - 風險評估
   - 受影響資產

#### 2.3 更新威脅狀態

1. 在威脅詳情頁面點擊「更新狀態」
2. 選擇新狀態（新增、分析中、已處理、已關閉）
3. 點擊「儲存」

### 3. 報告管理

#### 3.1 檢視報告清單

1. 點擊左側選單「報告管理」
2. 查看報告清單
3. 使用篩選功能查找特定報告

#### 3.2 生成報告

1. 點擊「生成報告」按鈕
2. 選擇報告類型（CISO 週報、IT 工單）
3. 設定時間範圍
4. 點擊「生成」

#### 3.3 下載報告

1. 在報告清單中點擊要下載的報告
2. 選擇下載格式（HTML、PDF、JSON）
3. 點擊「下載」

### 4. 系統設定

#### 4.1 檢視系統設定

1. 點擊左側選單「系統設定」
2. 選擇設定分類（威脅來源、通知規則、報告排程、資料保留）
3. 查看設定值

#### 4.2 更新系統設定

1. 在系統設定頁面選擇要更新的設定
2. 修改設定值
3. 點擊「儲存」
4. 確認變更

### 5. 稽核日誌

#### 5.1 檢視稽核日誌

1. 點擊左側選單「稽核日誌」
2. 查看稽核日誌清單
3. 使用篩選功能查找特定日誌

#### 5.2 匯出稽核日誌

1. 設定篩選條件
2. 點擊「匯出 CSV」或「匯出 JSON」
3. 下載匯出檔案

## 常見問題

### Q1：無法登入系統

**A**：請確認：
1. 帳號密碼正確
2. 帳號已啟用
3. 網路連線正常
4. 聯繫系統管理員

### Q2：無法匯入資產

**A**：請確認：
1. CSV 檔案格式正確
2. 檔案大小 ≤ 10MB
3. 必要欄位已填寫
4. 檔案編碼為 UTF-8

### Q3：報告生成失敗

**A**：請確認：
1. 時間範圍設定正確
2. 有足夠的資料
3. 系統資源充足
4. 聯繫系統管理員
